# Поисковые источники

## Подключено сейчас через `SEARCHAPI_KEY`

### 1. Yandex Reverse Image

Основной источник для поиска копий, похожих изображений и страниц, где изображение встречается.

Используется в режиме:

```text
Быстрый
Глубокий
Максимум
```

### 2. Google Lens — Exact Matches

Поиск точных совпадений через Google Lens.

Используется в режимах:

```text
Глубокий
Максимум
```

### 3. Google Lens — Visual Matches

Поиск визуально похожих изображений через Google Lens.

Используется в режимах:

```text
Глубокий
Максимум
```

### 4. Google Images Text Fallback

Дополнительный fallback: программа берёт связанные запросы из результатов и делает текстовый поиск по Google Images.

Используется в режиме:

```text
Максимум
```

## Что можно добавить позже

Эти источники требуют отдельных ключей и отдельной интеграции:

```text
TinEye API
DataForSEO Google Lens / Search by Image
SerpAPI Google Lens
Google Vision Web Detection
OpenAI Vision для описания изображения и генерации поисковых запросов
```

Важно: нельзя стабильно и легально собирать результаты Google/Яндекс/TinEye без официального API. Обход CAPTCHA, закрытых интерфейсов и авторизации в проект не добавляется.
