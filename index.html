Reverse Image Finder

Это репозиторий веб‑программы для обратного поиска изображений.

Важно: рабочая версия должна запускаться как Cloudflare Worker, потому что GitHub Pages не умеет безопасно хранить API‑ключи и выполнять backend‑код.

Как запустить

  1. Открой файл worker.js.
  2. Скопируй весь код в Cloudflare Worker.
  3. Добавь Secret SEARCHAPI_KEY.
  4. Создай Workers KV namespace и подключи его как TEMP_IMAGES.
  5. Нажми Deploy.

Подробности смотри в README и папке docs.
